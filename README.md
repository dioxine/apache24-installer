Пара скриптов для страдальцев от 1с, которым сложно было как и мне устанавливать последний Апач для 1с публикаций.
Инструкция:
1) скачиваем этот репозиторий с двумя батниками
2) распаковываем архив Apache24 скачанный последней сборки для винды x64 (если x86 сами правьте пути к програм файлз)
3) кладем папку распакованную Apache24 в одной директории с этими скриптами
4) запускаем install

успехов!
