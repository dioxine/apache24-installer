Пара скриптов для страдальцев от 1с, которым сложно было как и мне устанавливать последний Апач для 1с публикаций.
Инструкция:

1) скачиваем этот репозиторий с двумя скриптами
2) скачиваем с Apache Lounge например вот этот файл [httpd-2.4.66-260223-Win64-VS18.zip](https://www.apachelounge.com/download/VS18/binaries/httpd-2.4.66-260223-Win64-VS18.zip)
4) распаковываем из архива папку Apache24 в одну директорию с этими скриптами
5) запускаем install_apache24.bat - все
6) деинсталляция и очисткой следов деятельности - uninstall_apache24.bat

успехов!
